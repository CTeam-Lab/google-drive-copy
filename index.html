Instructions

  • Create a folder inside your Google Drive named
  • Move all the files you want to copy to your folder
  • Press 'Start Copy' and wait for feedback

Possible problems

  • The maximum runtime for Google Apps Script is 6 minutes, so if you have lots of files you might need to run this tool several times
  • If the file size is too large it might break the copy process due to Google restrictions
  • The script keeps track of the files that have been successfully copied to avoid duplicated files

Try it at your own risk!

  • Although I had no problems running this tool I am not responsible for any damages caused to the files during the copy process
